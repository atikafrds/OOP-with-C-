1. Folder bin berisi executable file dari kelima versi aplikasi
2. Folder doc berisi dokumen teknis, poster, log activity, dan pembagian kerja kelompok
3. Folder src berisi source code dari setiap versi aplikasi serta
dokumentasi doxygen dari versi s-1a
4. Setiap kelas yang dibuat (kecuali main program serta abstract base
class) disertai driver dengan nama file "m<nama kelas>.cpp"
5. Executable file hanya dapat dijalankan di linux
